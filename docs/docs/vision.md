# Vision

## 核心愿景 (Core Vision)
将 **Colony** 打造为一个 “三维空间 + 自动行为 + 观察驱动乐趣” 的挂机模拟游戏。

### 1. 三维空间 (3D Space)
- **Dwarf Fortress 风格的 Z-Level 系统**：游戏不再局限于平面，而是一个多层的垂直世界。
- **切片观察**：玩家每次观察世界的一个水平切片，通过切换不同高度来探索地下深处或高山之巅。
- **空间连通性**：单位在世界中寻找合适的高度生存，资源随高度分布。

### 2. 自动行为 (Autonomous Behavior)
- **自主个体**：每个单位（殖民者、动物、怪物）都有自己的动机（Hunger, Tired, Bored）。
- **任务驱动**：单位根据动机自主选择任务（Eat, Sleep, Chop, Forage），无需玩家频繁微操。
- **挂机乐趣**：玩家更像是一个观察者和规划者，看着小人们忙碌地建设和生活。

### 3. 观察驱动乐趣 (Observation-Driven Joy)
- **丰富的状态反馈**：通过头顶文字、状态面板，玩家可以直观了解到每个小人正在想什么、在做什么。
- **不可预测性**：性格系统和随机环境事件产生各种有趣的突发状况。
- **视觉沉浸感**：通过 2D 贴图在 3D 空间中的分层组合，创造出一种独特、深邃的像素世界观。
