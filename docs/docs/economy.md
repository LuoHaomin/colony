# Economy

## 资源闭环 (Resource Loop)
虽然当前版本处于早期阶段，但预设了以下资源逻辑：

### 1. 采集 (Collection)
- **伐木 (Woodcutting)**: 通过下达 `Chop` 指令，单位会自动寻找带有 `Choppable` 标签的植物。
- **采集 (Foraging)**: 单位可自主采集浆果、白菜等食物资源以维持生计。

### 2. 生长 (Growth)
- **季节系统 (`seasons.rs`)**: 环境温度与降水影响植物的生长速度和衰败。
- **种子与种植**: 单位可以执行 `Plant` 任务，将采集到的资源转化为地表的生长物。

### 3. 类型映射
- 资源类型详见 `src/core/objects.rs` 中的 `ItemType` 枚举。
- 每个物体都有其对应的材质、堆盈上限及用途定义。
