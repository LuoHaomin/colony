# 核心构思：观察者方舟 (Project Colony: The Observer's Ark)

## 1. 游戏定位 (Core Concept)
这是一款 **“深度模拟 × 自动化挂机”** 游戏。灵感来源于《环世界》(RimWorld) 和《矮人要塞》(Dwarf Fortress)，但玩家的角色从“微操者”转变为“观察者与政策制定者”。

## 2. 设计哲学 (Design Philosophy)
- **非直接控制**: 玩家不能命令某个小人去“砍那棵树”，但可以设定“木材阈值少于 100 时自动砍树”。
- **涌现行为 (Emergent Gameplay)**: 通过简单的底层规则（如：寒冷导致心情变差，心情变差导致斗殴）产生复杂的宏观故事。
- **深度切片**: 利用 Z-Level 设计，让世界在纵向上具有无限的探索和叙事空间。

## 3. 核心循环 (The Core Loop)
1. **观察 (Observe)**: 查看当前层级的单位状态、资源分布与环境威胁。
2. **分析 (Analyze)**: 理解为什么殖民地运转出现了隐患（例如：为什么大家都在挨饿？是因为没有田，还是因为没人愿意种？）。
3. **调控政策 (Tune Policies)**: 调整区域优先级、工作制度或资源分配方案。
4. **见证 (Witness)**: 观察系统在改动后如何自我修复，并产生新的动态平衡。
